# QuickCopy v3 — Action Wheel

**Variant:** v3 (radial / discoverable)
**Created:** 2026-04-24T19:34:46Z
**Status:** Proposal
**Adheres to:** ADR-0003 (Blok), ADR-0008 (shortcuts), ADR-0009 (errors), ADR-0010 (split button)

## 1. Design Direction

The Action Wheel reframes QuickCopy as a small radial console rather than a vertical list. Five copy actions are arranged in space so the eye can take in the whole tool in one glance: a labeled hub at the center anchors context (the page name and site), four satellite chips orbit it at diagonal positions, and the most-used action — Share Link — sits as a wide hero strip beneath the wheel.

The signature move is **discovery-by-hovering**: when the pointer or focus lands on a satellite, the hub temporarily replaces the page name with the actual value about to be copied (truncated URL, GUID, or title), shown in Geist Mono. Users learn what each glyph copies without clicking, and without a tooltip-heavy interface. This makes the wheel feel like a small instrument with live readouts rather than a button grid.

Visual character is calm and technical: thin 1px primary-tinted hairlines connect the hub to each satellite, a soft radial primary glow emanates from the wheel center, and there is no grain or texture. The wheel itself is static — only individual satellites animate on interaction. Typography is dual-tone: Geist Sans for human labels, Geist Mono for machine values (URLs, GUIDs, shortcut keys).

## 2. Layout & Structure

Panel: **320px wide, fits without vertical scroll**, pinned right against the mocked Pages canvas.

Vertical stack inside the panel (top to bottom):

| Region | Height | Contents |
|---|---|---|
| Header strip | 32px | "QuickCopy" wordmark left, theme toggle (sun/moon) right |
| Wheel area | 280px | 140px hub centered, four 64px satellites at diagonals, hairlines, radial glow |
| Share Link hero | 64px | Wide split-button: glyph `↑`, label "Share Link", caret right |
| Shortcut legend | 56px | Five Kbd chips in a row with sublabels (L Live, P Preview, I Item, T Title, S Share) |
| Footer caption | ~20px | Mocked Marketplace SDK note |

Wheel geometry: the hub is a 140px circle centered at the panel's horizontal midpoint. Satellites sit at 4 diagonal positions (top-left, top-right, bottom-left, bottom-right) at a radius of ~110px from hub center. Hairlines run from the hub edge to each satellite center. A Kbd chip rides on the outer edge of each satellite (offset diagonally outward) to keep shortcut letters discoverable without cluttering the chip face.

No element overlaps the hub during animations (scale ceiling 1.12, satellite radius leaves a 12px gap to hub edge at peak scale).

## 3. Components

**Hub (center)**
- 140px circle, `hsl(var(--background))` fill, 1.5px `hsl(var(--border))` stroke
- Two-line stacked content:
  - Page title in Geist Sans 600, auto-shrink from 16px → 13px when > 14 chars, single line, ellipsis at 18 chars
  - Site name below in Geist Mono 500, 10px, uppercase, letter-spacing 0.12em, `hsl(var(--muted-foreground))`
- On satellite hover/focus: content swaps to the **value preview** in Geist Mono 11px (URL truncated mid with `…`, GUID full, title full); reverts on leave
- On copy success: shows `Copied` in Geist Sans 600 14px for 1500ms, then reverts

**Satellite chip (×4: Live, Preview, Item ID, Title)**
- 64px circle, `hsl(var(--background))` fill, 1px `hsl(var(--primary))` border
- Glyph centered in Geist Sans 18px, `hsl(var(--primary))`: `↗` Live, `◉` Preview, `#` Item ID, `Aa` Title
- Kbd chip (18px tall) on outer diagonal edge: Geist Mono 10px, `hsl(var(--muted))` fill, `hsl(var(--muted-foreground))` text, 1px `hsl(var(--border))` border
- States:
  - Hover/focus: scale 1.1, fill becomes `hsl(var(--primary) / 0.12)`, glyph stays primary
  - Active (mousedown): scale 0.96
  - Copied morph: scale 1 → 1.12 → 1.0 over 250ms, glyph swaps to `✓` for 1500ms
  - Disabled (Live URL when unpublished): dashed 1px primary border, glyph `hsl(var(--muted-foreground))`, hairline becomes dashed, tooltip on hover/focus
  - Error (per ADR-0009): border becomes `hsl(var(--destructive))`, glyph swaps to `❌`, hairline turns destructive, persists until cache key changes

**Hairline (×4)**
- 1px solid `hsl(var(--border))` from hub edge to satellite edge
- Mirrors satellite state: dashed when disabled, destructive when in error

**Radial backdrop**
- Single `radial-gradient` centered on hub: `hsl(var(--primary) / 0.12)` at 0% → transparent at 70%
- No grain, no texture (deliberate distinction from v2)

**Share Link hero strip (split button per ADR-0010)**
- Full-width minus 16px padding, 56px tall, `hsl(var(--primary))` fill, `hsl(var(--primary-foreground))` text
- Layout: `↑` glyph | "Share Link" label (Geist Sans 600 14px) | flexible spacer | vertical divider | caret `▾`
- Primary region (left ~85%): copies Markdown by default (matches Alt+S)
- Caret region (right ~15%): opens dropdown with two items
  - "Copy as Markdown" (default, also Alt+S)
  - "Copy as Plain text"
- Hover: outer ring `0 0 0 4px hsl(var(--primary) / 0.18)`
- Copied: same scale-pulse morph as satellites; label swaps to "Copied" 1500ms

**Shortcut legend**
- 5-column grid; each cell: Kbd chip on top (Geist Mono 11px), sublabel below (Geist Sans 10px, `hsl(var(--muted-foreground))`)

**Demo controls (outside panel, top of page)**
- Three plain buttons: Toggle published / Force error: Preview URL / Reset states

## 4. Color & Typography

**Tokens (Blok, strict — no raw hex)**

Every color in the variant resolves through `hsl(var(--token))`. Tokens used: `--background`, `--foreground`, `--muted`, `--muted-foreground`, `--primary`, `--primary-foreground`, `--accent`, `--accent-foreground`, `--destructive`, `--destructive-foreground`, `--border`, `--ring`. Success states reuse `--primary` (Blok ships no dedicated success token).

Semantic mapping in this variant:

| Surface | Token |
|---|---|
| Panel & hub fill | `--background` |
| Body text, page title | `--foreground` |
| Site name, sublabels, value preview | `--muted-foreground` |
| Satellite border, hairline, glyph, Share Link fill | `--primary` |
| Share Link text | `--primary-foreground` |
| Hairline neutral state | `--border` |
| Error border, error hairline, error glyph | `--destructive` |
| Focus ring | `--ring` |
| Kbd chip fill | `--muted` |

**Fonts (Blok Nova preset, strict)**
- **Geist Sans** — UI labels, page title, button text, sublabels. Weights 400/500/600.
- **Geist Mono** — URLs, GUIDs, site name, Kbd chips, value previews. Weights 400/500.
- Loaded via Google Fonts: `https://fonts.googleapis.com/css2?family=Geist:wght@300..900&family=Geist+Mono:wght@400..700&display=swap`. No other fonts.

Type scale:

| Use | Font | Size | Weight |
|---|---|---|---|
| Page title (hub) | Geist Sans | 13–16px (auto) | 600 |
| Site name (hub) | Geist Mono | 10px upper +0.12em | 500 |
| Value preview (hub) | Geist Mono | 11px | 500 |
| Satellite glyph | Geist Sans | 18px | 600 |
| Share Link label | Geist Sans | 14px | 600 |
| Kbd chip | Geist Mono | 10–11px | 500 |
| Sublabel | Geist Sans | 10px | 500 |
| Header wordmark | Geist Sans tracking-tight | 12px | 600 |

## 5. Interaction States

**Idle.** Hub shows page title + site name. Satellites are outlined-primary. Hairlines neutral border color.

**Hover/focus on a satellite.**
- Satellite scales to 1.1, fill becomes faint primary tint
- Hub center swaps to value preview in Geist Mono
- Hairline does not change
- Reverts on `mouseleave` / `blur`

**Click / shortcut activation.**
- Write to clipboard via `navigator.clipboard.writeText`
- Copied morph: 250ms scale pulse on activated satellite (1 → 1.12 → 1.0)
- Glyph swaps to `✓` for 1500ms
- Hub center shows `Copied` in Geist Sans for 1500ms
- After 1500ms everything reverts; no toast

**Disabled (Live URL when page unpublished).**
- Dashed border, dimmed glyph, dashed hairline
- Hover/focus shows tooltip "Not published to Edge yet — publish the page first"
- Click and Alt+L are no-ops (return early, no morph, no error)

**Error (per ADR-0009).**
- Triggered by Force-error demo control or real clipboard rejection
- Satellite border → destructive, glyph → `❌`, hairline → destructive
- Hub shows `Failed` while satellite is hovered/focused
- **Persistent** until the value (cache key) changes; no auto-retry, no clear-on-leave
- Reset states demo control clears it

**Share Link.**
- Primary click → copy as Markdown, scale-pulse morph, label "Copied" 1500ms
- Alt+S → same as primary (does NOT open dropdown)
- Caret click → dropdown opens with two items, focus moves to first; Esc closes; Tab cycles within
- Selecting either item triggers the same morph

**Theme toggle.** Header sun/moon icon flips `.dark` on `<html>`, persists to `localStorage["quickcopy.poc.theme.v3"]`.

## 6. Accessibility

- **Contrast.** All foreground/background pairs pass WCAG AA (≥4.5:1) in both themes via Blok semantic tokens. Primary-on-primary-foreground (Share Link), foreground-on-background (hub), muted-foreground-on-background (sublabels) all verified.
- **Focus.** Every interactive element (satellites, Share Link primary, Share Link caret, dropdown items, theme toggle, demo buttons) shows a 2px `hsl(var(--ring))` outline with 2px offset. Focus order: theme toggle → satellite Live → Preview → Item ID → Title → Share Link primary → Share Link caret → legend (non-focusable).
- **Roles.** Satellites are `<button type="button">` with `aria-label` (e.g. "Copy Live URL — Alt+L"). Share Link split-button uses two adjacent buttons with shared `aria-describedby` per ADR-0010 (no ambiguous single-button-with-menu pattern). Dropdown is `role="menu"` with `role="menuitem"` children.
- **Live region.** A visually hidden `aria-live="polite"` region announces "Copied Live URL", "Copy failed for Preview URL", etc.
- **Keyboard shortcuts (ADR-0008).** `Alt+L/P/I/T/S` handled at panel scope only (the iframe in production). Suppressed when `event.target` matches `input, textarea, select, [contenteditable="true"]`. Shortcuts do not fire when focus is outside the panel.
- **Reduced motion.** `@media (prefers-reduced-motion: reduce)` collapses scale-pulse to a 1-frame opacity flash; Copied morph still announces via live region.
- **Screen reader value preview.** Hovering a satellite does not announce the preview (visual-only); the `aria-label` on the button already names the action. Copy success is announced via the live region.
- **Tooltip.** Disabled-state tooltip is reachable by keyboard focus, not just hover; uses `aria-describedby` linking to a visually hidden element.
