# QuickCopy — UI Design Spec v4

**Variant name:** Content-Forward Stack
**Generated:** 2026-04-24T19:34:46Z
**Design system:** Blok (ADR-0003), strict Nova preset
**Counterpoint to:** v3 (radial / decorative). v4 is restrained, content-forward, ruthlessly transparent.

---

## 1. Design Direction

**Concept.** Every action row literally previews the value it will copy. The user reads `Live URL → www.example.com/products/spring-campaign` before clicking, so the panel doubles as a manifest of the page's shareable identifiers. No abstract icons, no decorative glyphs as the focal point — the **value is the content**.

**Why this works.** Editors live in cognitive overload. v4 removes one decision per click: "what will I get if I copy this?" The answer is on screen, in monospace, at all times. Click anywhere on the row to copy.

**Atmosphere.** None. No gradients, no grain, no ambient glow, no radial wash. Visual rhythm comes from typography (Geist Sans display + Geist Mono values) and hairline borders. The "design" is the restraint.

**Personality.** Editorial, technical, opinionated. Closer to a CLI manifest than a marketing widget. The panel feels like a debug overlay that happens to be beautiful.

---

## 2. Layout & Structure

**Frame.** 320px sidebar pinned right against a darkened mocked Pages canvas (faint dot grid backdrop, caption "Mocked Pages editor canvas — not interactive"). Single column, no horizontal scroll, fits in viewport without vertical scroll.

**Vertical stack (top → bottom):**

| # | Section | Height | Purpose |
|---|---------|--------|---------|
| 1 | Header strip | 36px | Wordmark + theme pill |
| 2 | Page kicker | ~52px | Context: site · language, page title |
| 3 | Action rows × 4 | ~56px each | Live URL, Preview URL, Item ID, Page Title |
| 4 | Share Link row | ~64px | Primary-filled split-button (ADR-0010) |
| 5 | Legend | ~32px | Tight `L · P · I · T · S` shortcut row |

Total: ~420px panel height. Comfortable in the 320px sidebar canvas.

**Internal padding.** 16px horizontal gutters throughout. Rows align to the same left/right margins so the panel reads as a single typographic column.

## 3. Components

### 3.1 Header strip
- Left: `QuickCopy` wordmark, Geist Sans, weight 600, tracking-tight.
- Right: theme pill (`Dark` / `Light` text + Unicode sun/moon glyph), bordered, ~24px tall, rounded-full.
- Border-bottom: 1px `hsl(var(--border))`.

### 3.2 Page kicker
- Line 1: `EDITING` in Geist Mono uppercase, ~10px, tracking-widest, `hsl(var(--muted-foreground))`.
- Line 2: page title (`Spring Campaign Landing Page`) in Geist Sans semibold, ~15px, foreground.
- Line 3: `Northwind Marketing · en` in Geist Mono ~11px, muted-foreground.

### 3.3 Action row (Live URL / Preview URL / Item ID / Page Title)

Two-line structure:

```
LABEL                                 [K]
<value preview in Geist Mono, truncated>
```

- **Label:** Geist Sans uppercase, ~10px, tracking-wide (~0.08em), `hsl(var(--muted-foreground))`.
- **Kbd chip:** small bordered pill, Geist Mono ~10px, single letter (`L`, `P`, `I`, `T`). Border `hsl(var(--border))`, foreground muted.
- **Value preview:** Geist Mono ~12.5px, `hsl(var(--foreground))`, single line, `text-overflow: ellipsis`, `overflow: hidden`, `white-space: nowrap`. The killer detail.
- **Row container:** full-width clickable, border-bottom 1px `hsl(var(--border))`. No background in idle.
- **Hover:** background fills `hsl(var(--muted))`, label color promotes to foreground, border-left 4px `hsl(var(--primary))` slides in (transform).
- **Active flash:** brief 80ms primary-tinted flash, then value preview swaps to `Copied to clipboard ✓` in Geist Sans semibold, `hsl(var(--primary))`, for 1500ms. Reverts to original value on timeout.
- **Focus:** 2px focus ring `hsl(var(--ring))` offset 2px, visible only on `:focus-visible`.

### 3.4 Share Link row (special, primary-filled)

- Slightly taller (~64px), `hsl(var(--primary))` background, `hsl(var(--primary-foreground))` text.
- Left zone (primary action): `Share Link` label in Geist Sans semibold ~14px, with kbd chip `S` (inverse-tinted).
- Right zone before caret: small `MD` indicator, Geist Mono ~10px, in a subtle bordered chip showing the active format.
- Caret split (ADR-0010): vertical hairline divider, then a chevron button opens a dropdown.
- Dropdown menu items (each shows what the format produces):
  - `Markdown` — Geist Sans label, with Geist Mono preview line `[Title](url)` muted underneath.
  - `Plain text` — Geist Sans label, Geist Mono preview `Title — url` muted.
- Click primary zone copies in active format. Caret toggles dropdown. Selecting an item updates the `MD`/`TXT` chip and copies.

### 3.5 Legend strip
- Footer, `hsl(var(--muted))` background, ~32px tall.
- Five shortcuts in a tight row: each cell shows letter (Geist Mono uppercase) above a 9px Geist Sans muted action initial. Separators are middle-dots `·`.
- Format: `L · P · I · T · S` over `URL · PRV · ID · TTL · SHR`.

### 3.6 Demo controls (outside panel, top of page)
- Three buttons: `Toggle published`, `Force error: Preview URL`, `Reset states`.
- Bordered, `hsl(var(--muted))` background, Geist Sans 12px. Not part of the panel design.

## 5. Interaction States

### 5.1 Copy success
- Click row → 80ms primary flash → value preview swaps to `Copied to clipboard ✓` (Geist Sans semibold, `hsl(var(--primary))`) for 1500ms → reverts to original value. No toast.
- `aria-live="polite"` SR announcement: `<Action> copied to clipboard`.

### 5.2 Disabled (Live URL on unpublished)
- Value preview text becomes: `Not published to Edge yet — publish the page first` in `hsl(var(--muted-foreground))` italic.
- Row gets `aria-disabled="true"`, cursor `not-allowed`, no hover border-left, no click handler.
- Kbd chip dimmed.

### 5.3 Error (per ADR-0009 — persistent, no auto-retry)
- Value preview swaps to error message in `hsl(var(--destructive))`: e.g. `Couldn't fetch Preview URL — try switching pages or reloading`.
- Label gets `❌` prefix.
- Hover border-left becomes `hsl(var(--destructive))` instead of primary.
- Persistent until the cache key changes (page navigation or manual reset). Other rows unaffected.

### 5.4 Keyboard (ADR-0008)
- `Alt+L` Live URL · `Alt+P` Preview URL · `Alt+I` Item ID · `Alt+T` Page Title · `Alt+S` Share Link (Markdown default — does NOT open dropdown).
- Suppressed when focus is in `<input>`, `<textarea>`, or `[contenteditable]`.
- Iframe-scoped: handler bound to panel document, no global window pollution.

### 5.5 Theme toggle
- Pill click toggles `.dark` on `<html>`, persisted to `localStorage['quickcopy.poc.theme.v4']`. Default = dark.

## 6. Accessibility

- **Contrast.** All text/background pairs verified WCAG AA in both themes. Foreground-on-background and primary-foreground-on-primary both ≥ 4.5:1 for normal text, ≥ 3:1 for ≥14px semibold.
- **Focus.** All interactive elements have visible `:focus-visible` ring (`hsl(var(--ring))`, 2px offset). Tab order: theme pill → action rows top-to-bottom → share primary → share caret → demo controls.
- **Roles.** Action rows are `<button>` elements. Disabled rows use `aria-disabled="true"` (still tab-focusable for discoverability). Share dropdown uses `role="menu"` + `role="menuitem"`; caret has `aria-haspopup="menu"` + `aria-expanded`.
- **Live region.** Single `aria-live="polite"` SR-only node announces copy success and errors.
- **Reduced motion.** `@media (prefers-reduced-motion)` disables border-left slide and flash transitions.
- **Truncation tooltips.** Each row's value preview has `title` attribute with the full untruncated value, so screen-reader and hover users see the complete text.
